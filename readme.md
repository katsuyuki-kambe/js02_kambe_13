①課題内容（どんな作品か）
凸凹道を走るバイクゲームをyoutubeを参照に作成しました。


②工夫した点・こだわった点
１）難易度が高くなりすぎないような凸凹の調整や十字キーによる操作
　上キー：前進　下キー：バック　右キー：前傾　左キー：後傾
２）経過秒数をカウントできるように設定
３）localhostに前回のタイムを保存・呼び出し（手動）

＜反省点と今後のブラッシュアップ予定＞
ゲームオーバー、リスタート機能のJSでの設定、
タイムのlocalhostへの自動保存（そもそもゲームの終わりの設計ができていなかった）、
ゲームっぽい華やかな画面、background設定が上手くできていない。
バイクゲームがなかなか動かず、Localhost周りの実装が雑になってしまった。
youtubeのタイトルは１時間で作れるとなっていましたが、はしょられている部分が多かったので、実際は１日かかりました。
